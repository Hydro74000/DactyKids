# Guide parent / enseignant

## Sessions

- Viser des sessions de 3 a 8 minutes.
- Chercher la precision avant la vitesse.
- Faire une pause quand les erreurs se repetent.

## Lecture des resultats

- Une lecon terminee a 90 % ou plus est consideree acquise.
- Les touches a retravailler indiquent les erreurs recurrentes du profil actif.
- Le WPM reste indicatif: il sert surtout aux enfants deja a l'aise.

## Parcours

- Village: rangee de repos.
- Foret: rangee superieure.
- Grotte: rangee inferieure.
- Jardin: mots et phrases simples.
- Chateau: majuscules, chiffres, ponctuation et paragraphes courts.
- Accents: exacts sur AZERTY francais, simplifies sur QWERTY.

## Donnees locales

- Les profils sont separes localement.
- Aucun compte enfant n'est requis.
- Le CSV parent peut etre colle dans un tableur pour un suivi ponctuel.
- La sauvegarde JSON parent restaure les profils et les progressions locales.
- Un profil enfant peut etre supprime localement avec sa progression; au moins un profil reste toujours disponible.

## Audio

- Le reglage Son doux active des sons courts locaux.
- Les sons signalent une touche juste, une correction et une fin de lecon.
- Aucun son n'est lu quand le reglage est desactive.

## Accessibilite

- Le mode Tres lisible augmente la typographie de l'application.
- Le mode Niveaux de gris reduit les indices dependants de la couleur.
- Les controles principaux gardent une cible confortable.
- L'interface est testee avec un scaling systeme a 200 % sur les ecrans principaux.
- Les actions principales de l'accueil sont atteignables au clavier avec Tab et Entree.
